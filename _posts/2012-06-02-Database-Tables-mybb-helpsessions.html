Stores the categories of help documents.

Field Type Null Key Default Description
sid smallint(5) No Primary Section ID
name varchar(120) No Section name
description text No Section description
usetranslation char(3) No 'yes'/'no' to use the translation from language files
enabled char(3) No 'yes'/'no' to enable the section
disporder smallint(5) No 0 Display order
